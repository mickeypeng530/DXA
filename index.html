⑦ TBS + FRAX
條件做法
男性 ≥ 50歲 / 女性停經/亂經做 TBS + FRAX
男性 < 50歲 / 女性未停經不做
90歲以上不做
L spine 全部排除只做 FRAX
雙側 hip 術後只做 TBS
骨強度 = 骨量 BMD（2/3）+ 骨質結構 TBS（1/3）

FRAX 風險因子（詳見附錄）：

  • 過去骨折史（成年後自發性骨折）
  • 父母髖骨骨折
  • 目前吸菸
  • 類固醇（≥3個月，≥5mg/day）
  • 類風濕性關節炎
  • 每日飲酒 ≥ 3 單位
  • 續發性骨鬆：糖尿病、甲狀腺亢進、早發停經（<45歲）、慢性肝病、吸收不良
附錄

補充教學內容，點擊展開

定義：一種讓骨骼強度減弱、增加骨折風險的疾病

  • 40歲後：骨密度每年 −0.5 ~ −1%
  • 70歲後：每年 −1 ~ −2%

脆弱性骨折：低衝擊力下骨折（手腕、肱骨、骨盆、脊椎、髖骨）。車禍不算。

  • 高能量 X 光：穿透力強（吸收少）
  • 低能量 X 光：骨頭吸收多
  • 所有掃描點 → 2D 圖 → 骨點/非骨點 → 骨質量
  • 單位：g/cm²｜輻射：約 0.001 mSv
T = (受檢BMD − 年輕參考BMD) / 年輕參考SD
Z = (受檢BMD − 同齡參考BMD) / 同齡參考SD
T-Score診斷
≥ −1.0正常
−1.0 ~ −2.5低骨量（Low bone mass）
≤ −2.5骨質疏鬆（Osteoporosis）
TBS意義
> 1.311Normal
1.231 ~ 1.311Partially degraded
≤ 1.231Degraded

續發性骨質疏鬆症：

若患者有以下疾病，選擇「是」：
  • 糖尿病（尤其胰島素依賴型）
  • 成骨不全症（Osteogenesis Imperfecta）
  • 長期未治療的甲狀腺機能亢進
  • 性腺功能低下症
  • 早發性停經（小於 45 歲）
  • 慢性營養不良
  • 吸收不良症候群（如乳糜瀉）
  • 慢性肝臟疾病
